Block roles

Control–observation duality. GDS's fourth role, ControlAction, is the output map y = C(x). The same map an external controller reads as its action is what an actor inside the model reads as an observation — so our read-out block is named Observe (the actor's view) yet carries the ControlAction role (the controller's view). One map, two perspectives; which name you pick is a matter of whose frame you take.
