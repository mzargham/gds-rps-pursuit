Rock-paper-scissors pursuit-evasion

A toy two-team adversarial dynamic game built on the Generalized Dynamical Systems formalism. Each asset is a rock, paper, or scissors; team leaders assign tasks; assets move under one symbolic energy field; contact resolves by rock-paper-scissors. This is a replay of a single recorded match.

Demonstration only — a toy world illustrating the GDS methodology (a simulation generated directly from a symbolic spec), not a realistic or solved pursuit model. See the formal structure page and the README for the simplifying assumptions.
